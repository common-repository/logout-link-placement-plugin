Logout Link Placement Plugin allows you to place the logout link in the left navigation of the administration panel.

== Description ==

This plugin adds a **logout link** in the left navigation of the administration panel to allow you to log out.

[Support](http://kshitijahuja.com/wordpress-plugins/logout-link-placement-plugin) |
[Change Log](http://kshitijahuja.com/wordpress-plugins/logout-link-placement-plugin) |
[FAQ](http://kshitijahuja.com/wordpress-plugins/logout-link-placement-plugin)|

Some features:

* Allows you to easily log out using the link in the left navigation of the administration panel.
* Easy to install and use


== Installation ==

You can use the built in installer and upgrader, or you can install the plugin
manually. For manual installs,

1. Upload the 'logout-link-placement-plugin' folder into your wp-content/plugins directory. 
2. Go to Plugins menu in the Wordpress Adminstration and you will find the plugin listed. Click Activate to activate the plugin.
3. You will see a 'Log out' navigation item in the left menu in the administration panel. 
4. That's it!
